# 🎮 Fairy Games

An interactive and aesthetic web-based game platform featuring a dynamic quiz game built using **HTML, CSS, and JavaScript**.

---

## ✨ Features

* 🎯 Interactive Quiz Game
* ⚡ Instant Answer Feedback (Correct / Wrong)
* 🔄 Automatic Next Question Transition
* 🧠 Multiple Questions with Score Tracking
* 🎨 Aesthetic UI with soft fairy theme
* 📱 Responsive layout using Flexbox & Grid

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Layout (Flexbox + Grid)
* **JavaScript (Vanilla)** – Game Logic & Interactivity

---

## 🚀 How It Works

1. User selects an answer
2. Clicks **Submit**
3. Instant feedback is shown
4. Automatically moves to next question
5. Final score is displayed at the end
